# LeetCode 大挑戰

簡介：LeetCode 的總題數超過 2,000 題，在時間有限的情況下，挑選最常見的題型，進行刷題練習。

在實作過程中，發現多數演算法的最佳解，皆根基於數學原理，是非專業背景者想破頭也想不出來的。

因此我的練習過程，會分為三階段：

1. 暴力破解：使用最直覺的方法解題，並且在解題過程中，思考如何優化。
2. 演算法題解：參考 Solution 的作法，嘗試理解箇中原理，並盡可能將解法背起來。
3. 重新演練：相隔一段時間後，重新挑戰同一題目，看自己是否能獨立解題。
